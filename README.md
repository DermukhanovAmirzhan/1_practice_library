# Көлік құралдарын басқару жүйесі

## Сипаттама

Бұл жоба Java тілінде объектіге бағытталған бағдарламалау (OOP) принциптерін пайдаланып жасалған.  
Жоба көлік құралдарын басқаруды модельдейді және келесі ұғымдарды көрсетеді:

- Мұрагерлік (Inheritance)  
- Композиция (Composition)  
- Полиморфизм (Polymorphism)  

## Жоба құрылымы

- `Vehicle` — базалық класс  
- `Car`, `Motorcycle` — туынды класстар  
- `Garage` — көлік құралдарының жиыны  
- `Fleet` — гараждардың жиыны  

## Технологиялар

- Java 17+  
- IntelliJ IDEA Community Edition  
- OOP принциптері  

## Іске қосу

1. IntelliJ IDEA-да жобаны ашу  
2. `Main.java` файлын іске қосу  
3. Код автопарк, гараждар және көлік құралдарының жұмысын көрсетеді
